# 🌾 Panchayat Portal

A modern **MERN Stack** web application that digitizes essential village services by providing citizens with a single platform to access local service providers, emergency contacts, community events, and grievance management.

## 🚀 Live Demo

🌐 **Website:** https://panchayat-portal-rho.vercel.app/

## 💻 GitHub Repository

https://github.com/Omkar0940/panchayat-portal

---

# 📖 About the Project

Panchayat Portal is a full-stack web application built to simplify access to village-level services through a responsive and user-friendly digital platform.

The platform enables citizens to:

- Find nearby service providers
- View upcoming community events
- Register complaints online
- Access emergency contacts instantly
- Manage user accounts securely
- Use an AI-powered assistant for guidance

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Omkar0940/panchayat-portal.git
```

Go to the project folder

```bash
cd panchayat-portal
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file

```env
PORT=3001

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

Run backend

```bash
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
